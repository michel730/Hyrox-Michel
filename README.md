# Carnet Prépa HYROX Michel V4

Pack prêt pour GitHub Pages.

## Mise en ligne
1. Créer un dépôt GitHub nommé `hyrox-michel`.
2. Décompresser ce ZIP.
3. Envoyer `index.html`, `manifest.json`, `service-worker.js`, `icon-192.png`, `icon-512.png` et `README.md` à la racine du dépôt.
4. GitHub > Settings > Pages.
5. Source : Deploy from a branch.
6. Branch : main, folder : /root.
7. Ouvrir l’URL GitHub Pages dans Safari sur l’iPhone.
8. Partager > Ajouter à l’écran d’accueil.
